### 요약
- 뉴로 마케팅을 활용한 광고 분석 프로그램.

- 광고를 시청하는 동안의 사용자의 집중도와 긍부정도가 그래프로 나타남.

- 실험에 기반한 계산을 통해 사용자의 집중도와 긍부정도를 고려한 구매 확률 제시
